How to Use

  1. Adjust array size and speed using the sliders
  2. Generate a new random array or shuffle the current one
  3. Select a sorting algorithm from the available options
  4. Click "Start Sorting" to visualize the algorithm in action
  5. Observe color changes highlighting comparisons, swaps, and sorted positions
  6. Pause or reset the visualization at any time
