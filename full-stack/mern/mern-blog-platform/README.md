# MERN Blog Platform

A modern blogging platform for developers to share their knowledge.

## Project Structure
- `client/`: React frontend (Next.js/React, Markdown support)
- `server/`: Node.js backend (Express, Cloudinary for images)

## Features
- Rich Text Editor
- Image Uploads (Cloudinary/S3)
- Comments and Likes system
- SEO optimized pages
- User dashboard for analytics

## Setup Instructions

### Backend (Server)
1. `cd server`
2. `npm install`
3. Set up `.env` for MongoDB and Cloudinary.
4. `npm start`

### Frontend (Client)
1. `cd client`
2. `npm install`
3. `npm run dev`
