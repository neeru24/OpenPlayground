# MERN Task Manager

A collaborative task management tool for teams.

## Project Structure
- `client/`: React frontend (Drag & Drop, Charts)
- `server/`: Node.js & Express backend (Task scheduling, MongoDB)

## Features
- Kanban Board interface
- Task Assignment & Deadlines
- Real-time updates
- Project milestones
- Productivity charts

## Setup Instructions

### Backend (Server)
1. `cd server`
2. `npm install`
3. Configure database connection in `.env`.
4. `npm run dev`

### Frontend (Client)
1. `cd client`
2. `npm install`
3. `npm start`
