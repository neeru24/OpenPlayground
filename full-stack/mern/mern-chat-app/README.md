# MERN Real-Time Chat App

A real-time messaging application using Socket.io and the MERN stack.

## Project Structure
- `client/`: React frontend (Socket.io-client, Framer Motion)
- `server/`: Node.js & Express backend (Socket.io, MongoDB)

## Features
- Real-time messaging with Socket.io
- Individual and Group Chats
- File sharing capabilities
- Online/Offline status
- User Profiles

## Setup Instructions

### Backend (Server)
1. `cd server`
2. `npm install`
3. Configure MongoDB settings in `.env`.
4. `npm run dev`

### Frontend (Client)
1. `cd client`
2. `npm install`
3. `npm start`
