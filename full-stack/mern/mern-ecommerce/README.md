# MERN E-Commerce Store

A full-featured e-commerce platform built with MongoDB, Express, React, and Node.js.

## Project Structure
- `client/`: React frontend (Vite, Redux, Tailwind)
- `server/`: Node.js & Express backend (MongoDB, JWT)

## Features
- User Authentication (JWT)
- Product Search & Pagination
- Shopping Cart & Checkout
- Admin Dashboard for Product Management
- Payment Integration (Stripe/PayPal)

## Setup Instructions

### Backend (Server)
1. `cd server`
2. `npm install`
3. Create `.env` file with `MONGO_URI` and `JWT_SECRET`.
4. `npm start`

### Frontend (Client)
1. `cd client`
2. `npm install`
3. `npm run dev`
